사용자 A가 작성한 파일입니다

1. 계정 정보 등록
	git config - global user.name 'goand123'
	git config - global user.email 'goand321@naver.com'
2. userA 폴더를 git이 관리하는 폴더로 만들기
	git init
3. 기본으로 생성되는 master 브랜치를 main 브랜치로 바꿔주자
	git checkout -b main : main이라는 이름의 브랜치를 생성하고 해당 브랜치로 이동
	git branch -M main : 기본브랜치를 main 브랜치로 지정
						(git에서 git-hub로 올릴때 잘못된 브랜치가 생성될수있어)
	git status : 현재상태 확인
4. 파일을 생성하고 add, commit 하기
	git add <파일명> : 파일을 staging area 추가
	git commit : commit에 대한 메세지를 남기기 위한 vi 에디터가 실행된다.
				i : 입력모드, esc : 입력모드 빠져나오기, wq : 저장 종료
	git commit -m 'message' : commit을 하고 commit에 대한 메세지를 남긴다.
	commit 내역은 git log 명령어를 통해 확인 가능
5. 파일 특정 commit 지점으로 되돌리기
